Course Curriculum

This module explains the basics of graphic design, including design principles, design elements, typography, color theory, color combinations, layouts, and branding fundamentals.

This module teaches image editing and design using Adobe Photoshop, including photo retouching, color correction, layers, selections, and creating posters, banners, and social media designs.

This module focuses on creating professional layouts using Adobe InDesign, including brochures, magazines, flyers, typography handling, page layouts, and print-ready documents.

This module covers vector graphic design using Adobe Illustrator, including logo design, icon creation, illustrations, pen tool usage, color management, and exporting vector files.

This module introduces UI and UX design using Figma, including wireframing, interface design, prototyping, responsive layouts, and team collaboration.

This module teaches quick and easy design creation using Canva, including social media posts, presentations, posters, brand kits, and marketing creatives.

This module focuses on vector graphic design using Inkscape, including drawing tools, path operations, logo creation, typography, and exporting SVG and print files.
